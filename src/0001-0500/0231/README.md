### 2的幂

给你一个整数 n，请你判断该整数是否是 2 的幂次方。如果是，返回 true ；否则，返回 false 。

如果存在一个整数 x 使得 n == 2x ，则认为 n 是 2 的幂次方

### 示例

```
输入：n = 1
输出：true
解释：20 = 1

输入：n = 16
输出：true
解释：24 = 16

输入：n = 3
输出：false

输入：n = 4
输出：true

输入：n = 5
输出：false
```

### 提示

- `-2^31 <= n <= 2^31 - 1`
- 你能够不使用循环/递归解决此问题吗？

### 常用位运算操作

`n&(n-1)` 这个操作是算法中常见的，作用是消除数字 n 的二进制表示中的最后一个 1

```
n:       110100
n-1:     110011
n&(n-1): 110000
```

核心逻辑就是，`n-1` 一定可以消除最后一个 1，同时把其后的 0 都变成 1，这样再和 n 做一次 `&` 运算，就可以仅仅把最后一个 1 变成 0 了
